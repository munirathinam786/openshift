Security Console

Dedicated React workspace for OpenShift audit, SCC review, OAuth / LDAP posture, GitOps and Tekton delivery controls, cluster logging, OADP, ACM governance, ACS coverage, network policy posture, quota governance, operator lifecycle oversight, and baremetal / ROSA / ARO / IBM Z platform-pattern reviews.

  • Choose audit profiles like cluster hardening, namespace governance, GitOps / Tekton delivery, logging and backup posture, ACM fleet posture, ACS coverage, ingress exposure, storage posture, and platform-pattern review.
  • Select the OpenShift security controls to include in the review, including OAuth / LDAP, GitOps, Tekton, logging, OADP, ACM, and ACS objects.
  • Generate live exportable reviews from the dedicated POST /security/audit endpoint built for batched compliance runs.
