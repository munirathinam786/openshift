Platform Console

Dedicated React workspace for platform lifecycle readiness, DR and failover planning, OpenShift Virtualization / CNV posture, migration reviews, and fleet-aware day-2 engineering checks.

  • Choose a profile like lifecycle readiness, DR posture, migration factory, virtualization, or fleet posture.
  • Select the exact OpenShift platform signals to inspect, including cluster version, machine pools, ACM, OADP, CNV, DR resources, storage, logging, GitOps, and platform inventory.
  • Run the review through the live POST /chat endpoint and export the resulting platform summary for operator handoff.
