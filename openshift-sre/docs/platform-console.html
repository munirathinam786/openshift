Platform Console

Dedicated React workspace for platform lifecycle readiness, DR and failover planning, OpenShift Virtualization / CNV posture, migration reviews, workload disruption safety, and fleet-aware day-2 engineering checks.

  • Choose a profile like lifecycle readiness, DR posture, migration factory, virtualization, or fleet posture.
  • Select the exact OpenShift platform signals to inspect, including cluster version, machine pools, cluster networking, ingress controllers, autoscaling, disruption budgets, RBAC, BuildConfigs, DeploymentConfigs, ACM, OADP, CNV, DR resources, storage, logging, GitOps, Knative, MTC, and platform inventory.
  • Run the review through the live POST /chat or POST /chat/stream endpoint, then compare history, build watchlists, or export the resulting platform summary for operator handoff.
