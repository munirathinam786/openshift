# Draw.io Diagrams Index

All architecture, cluster, pipeline, and code diagrams are available as interactive Draw.io files and rendered SVG images. Download any `.drawio` file and open it in [draw.io](https://app.diagrams.net) for full interactive editing.

!!! tip "How to use these diagrams"
    - **SVG images** are embedded directly in the documentation pages
    - **`.drawio` files** can be downloaded and opened in the [draw.io desktop app](https://github.com/jgraph/drawio-desktop/releases) or [draw.io online](https://app.diagrams.net) for interactive editing
    - All original Mermaid source is preserved in collapsible sections on each page
