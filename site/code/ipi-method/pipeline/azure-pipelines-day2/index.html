    azure-pipelines-day2.yml — IPI Day 2 Pipeline¶

    Azure DevOps pipeline definition for IPI Day 2 post-install operations (Cluster Logging, OADP, LDAP/OAuth, GitOps/ArgoCD, Pipelines/Tekton).

    See the pipeline documentation for parameter details and usage.

    Source Code¶

    # =============================================================================
# Azure DevOps Pipeline — Day 2 Operations (Post-Install)
# Cluster Logging, OADP Backup/Restore, LDAP/OAuth, GitOps (ArgoCD), Pipelines (Tekton)
